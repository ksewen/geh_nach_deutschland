# Start auf Deutsch
## Begrüßungen
* *Guten Morgen.*
* *Guten Tag.*
* *Guten Abend.*
* *Gute Nacht.*

## Fragen und Antworten
### Name
```
+ Wie heißen Sie?
- Ich heiße Reus.
```

```
+ Wie ist Ihr Name?
- Mein Name ist Reus.
```

```
+ Wer sind Sie?
- Ich bin Reus.
```
### Herkunft
```
+ Woher kommen Sie?
- Ich komme aus Deutschland.
```

```
+ Woher kommst du?
- Ich komme aus Deutschland.
```

```
+ Woher sind Sie?
- Ich bin aus Deutschland.
```

```
+ Woher bist du?
- Ich bin aus Deutschland.
```
### Wohnort
```
+ Wo wohnen Sie?
- Ich wohne in Dortmund.
```

```
+ Wo wohnst du?
- Ich wohne in Dortmund.
```
